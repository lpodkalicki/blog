Release Notes for STM32091C-EVAL Evaluation Board Drivers

Copyright 2016 STMicroelectronics

 

Update History

V2.0.7 / 25-August-2017

Main Changes

  • Minor updates in STM32091C_EVAL_BSP_User_Manual.chm documentation.

V2.0.6 / 7-April-2017

Main Changes

  • Updated BSP driver documentation.

V2.0.5 / 04-November-2016

Main Changes

  • Removed BSP/CEC driver : refer to to HAL for CEC.

V2.0.4 / 27-May-2016

Main Changes

  • Aligned BSP drivers to use latest BSP components.

V2.0.3 / 29-January-2016

Main Changes

  • Updated GPIO Output Speed literals naming to ensure HAL full compatibility.
  • Added new services for SD management over SPI.
  • Added RX Fifo Flush at end of LCD_IO_WriteMultipleData().

V2.0.2 / 26-June-2015

Main Changes

  • Complete HAL API alignment (renaming).
  • Replace HAL_CEC_ErrorTypeDef by uint32_t.
  • Add CHM User Manual for BSP/STM32091C_EVAL board.
  • Implement workaround to cover RCC limitation regarding peripheral enable delay.

V2.0.0 / 03-Oct-2014

Main Changes

  • Correct compilation warning with TrueSTUDIO & typo in defgroup in stm32091c_eval_cec.c/.h

V2.0.0RC1 / 01-Aug-2014

Main Changes

  • First version of the STM32091C-EVAL evaluation board drivers.
  • This version has to be used only with STM32CubeF0 based development
